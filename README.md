# jquery gridsystem

## Integration

## Usage
Just add the following hashes to your URL and the grid system overlay will be displayed:

<ul>
	<li>#grid</li>
	<li>#gridx</li>
	<li>#gridy</li>
</ul>

	Example: http://example.com/#grid
or, for column view:
	Example: http://example.com/#gridx
